# Prince Phones E-commerce Site

Welcome to the Prince Phones online store! This is a responsive and user-friendly e-commerce website designed to showcase and sell a variety of smartphones.

## Features

- **Responsive Layout:** Optimized for desktops, tablets, and mobile devices using CSS Grid and media queries.
- **Product Listings:** Attractive product cards with images, badges (Sale, New, Bestseller), descriptions, and prices.
- **Animations and Transitions:** Smooth fade-in effects for products, button hover animations, and badge pop-ins to enhance user experience.
- **Accessible Design:** Semantic HTML structure and ARIA labels for better accessibility.
- **SEO Optimized:** Schema.org structured data for products and offers to improve search engine visibility.
- **User Interaction:** Interactive buttons with hover effects and clear call-to-actions.
- **Clean and Modern UI:** Stylish color schemes and layouts that create an engaging shopping experience.

## Technologies Used

- HTML5 for semantic markup
- CSS3 with Grid, Flexbox, and animations
- Optional JavaScript for enhanced interactivity (to be added)
- Responsive design principles
